# 📊 Nutri Stats - Dashboard

O **Dashboard do Nutri Stats** é a parte principal da aplicação, onde o usuário pode visualizar, organizar e acompanhar informações relacionadas à sua alimentação e dados nutricionais.

## 🚀 Tecnologias utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização e responsividade
* **JavaScript** → Lógica, interações e manipulação de dados

## 💡 Sobre o projeto

O dashboard foi desenvolvido com o objetivo de oferecer uma interface clara e funcional para:

* Visualizar dados alimentares
* Acompanhar hábitos nutricionais
* Organizar informações de forma prática

Ele representa a evolução da landing page, trazendo mais interatividade e funcionalidades para o usuário.

## 🎯 Funcionalidades

* 📌 Visualização de dados nutricionais
* 🥗 Listagem de alimentos cadastrados
* 📊 Organização de informações em formato intuitivo
* 🔍 Interface simples e de fácil navegação
* 📱 Layout responsivo para diferentes dispositivos

## 🧠 Estrutura do sistema

O dashboard foi estruturado pensando em escalabilidade, permitindo futuras integrações com:

* Banco de dados
* APIs externas
* Sistema de autenticação

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados:

* Manipulação de DOM com JavaScript
* Organização de código front-end
* Criação de interfaces modernas e responsivas
* Pensamento em experiência do usuário (UX)

## 🔧 Melhorias futuras

* Integração com backend e banco de dados
* Sistema de login e autenticação
* Gráficos dinâmicos (calorias, nutrientes, etc.)
* Filtros e busca de alimentos
* Histórico alimentar do usuário

## 📌 Observação

Este dashboard ainda está em desenvolvimento e faz parte do projeto maior **Nutri Stats**, que tem como objetivo se tornar uma plataforma completa de acompanhamento nutricional.
